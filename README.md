# MERN Boilerplate

## Description

Simple app with React, Redux, Typescript, React-router and Express for API.

```sh
./frontend - React app
./backend - Express.js server
```

## Scripts

```sh
npm run server
```

Start express server separately

```sh
npm run client
```

Start React app separately

```sh
npm run start:dev
```

Start React app with Express server together
